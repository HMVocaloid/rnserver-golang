Publisher's Description

The official DivX video software bundle, this download contains the software you need to watch and create videos in the popular DivX video format. The DivX 5.1 generation includes many advanced features, such as enhanced video quality, a feedback mode, more effective compression capabilities, and full support for decoding MPEG-4 Advanced Simple Profile video. DivX also is optimized for AMD Athlon and Athlon XP processors. This bundle includes the DivX Player 2.5, which supports the MPEG-4 format and offers a skinnable interface system.

Version 5.1.1 features support for generic MP4 FourCC codes, which can be enabled or disabled in the DirectShow configuration dialog box.
